Buffer & Clip Site

Cara pakai:

  1. Klik tombol Site via Koordinat di dalam peta untuk memasukkan lokasi dengan koordinat atau gambar titik/garis/polygon langsung dari toolbar.
  2. Isi jarak buffer (Isi 0 km jika shape area ingin dipakai langsung sebagai area clip)
  3. Klik Analisis dari Koordinat di pop-up peta atau Jalankan Buffer untuk menghimpun layer di sekitar site.
  4. Unduh paket ZIP hasil clip dan lanjutkan analisis di ArcGIS.
